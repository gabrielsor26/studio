# Historias de Usuario - Proyecto "Misty Box"

Este documento centraliza todas las historias de usuario que definen las funcionalidades de la plataforma "Misty Box", sirviendo como guía para el desarrollo y la validación.

| ID | Épica | Rol de Usuario | Historia de Usuario (Yo como... quiero... para...) | Criterios de Aceptación |
| :--- | :--- | :--- | :--- | :--- |
| **HU-01** | **Descubrimiento** | Visitante | Como **visitante**, quiero **ver una lista de todas las oportunidades laborales** para **explorar lo que ofrece la plataforma sin registrarme**. | - La página de inicio muestra una lista de oportunidades.<br>- Puedo buscar ofertas por texto (título, empresa, descripción).<br>- Puedo filtrar las ofertas por sector (Público, Privado).<br>- Al hacer clic en una oferta, soy redirigido a su página de detalle. |
| **HU-02** | **Descubrimiento** | Visitante | Como **visitante**, quiero **ver el detalle de una oportunidad laboral** para **entender todos los requisitos y beneficios antes de decidir postular**. | - La página muestra el título, empresa, modalidad, descripción, requisitos y condiciones de la oferta.<br>- Se diferencia visualmente si la oferta es del sector público o privado.<br>- Se muestra un botón claro para iniciar el proceso de postulación. |
| **HU-03** | **Descubrimiento** | Visitante | Como **visitante**, quiero **explorar un directorio de empresas** para **conocer qué organizaciones forman parte de la plataforma**. | - Se muestra una lista de empresas con su logo, nombre y descripción.<br>- Puedo buscar empresas por nombre.<br>- Al hacer clic en una empresa, soy redirigido a su perfil público. |
| **HU-04** | **Descubrimiento** | Visitante | Como **visitante**, quiero **ver el perfil público de una empresa** para **saber más sobre su cultura y las oportunidades que ofrece**. | - La página de perfil de empresa muestra su logo, nombre, descripción, sector y sitio web.<br>- Se muestra una lista de las ofertas de trabajo activas publicadas por esa empresa. |
| **HU-05**| **Autenticación** | Nuevo Usuario (Postulante) | Como un **nuevo postulante**, quiero **crear una cuenta** para **poder postular a las ofertas de trabajo**. | - Puedo acceder a un formulario de registro desde la página de login.<br>- El formulario me pide nombre, email y contraseña.<br>- Al registrarme exitosamente, soy autenticado y redirigido a mi perfil. |
| **HU-06**| **Autenticación** | Nuevo Usuario (Empresa) | Como una **nueva empresa**, quiero **crear una cuenta de empresa** para **publicar y gestionar mis propias ofertas laborales**. | - Puedo acceder a un formulario de registro para empresas.<br>- El formulario me pide nombre de la empresa, email y contraseña.<br>- Al registrarme, soy autenticado y redirigido al dashboard de empresa. |
| **HU-07**| **Autenticación** | Usuario Registrado | Como un **usuario registrado**, quiero **iniciar sesión** para **acceder a mi perfil y funcionalidades personalizadas**. | - Puedo iniciar sesión con mi email y contraseña.<br>- Si el login es exitoso, soy redirigido a mi página principal (Perfil para postulantes, Dashboard para empresas).<br>- Si las credenciales son incorrectas, recibo un mensaje de error. |
| **HU-08**| **Postulaciones** | Postulante | Como **postulante**, quiero **postular a una oferta del sector privado** para **enviar mi CV directamente a la empresa**. | - Al hacer clic en "Postular", se abre un modal si estoy autenticado.<br>- El modal me permite adjuntar un archivo (CV en PDF).<br>- Al enviar, recibo una notificación de éxito (`toast`). |
| **HU-09**| **Postulaciones** | Postulante | Como **postulante**, quiero **ser guiado para postular a una oferta del sector público** para **completar mi postulación en el portal oficial**. | - Al hacer clic en "Postular", un modal me informa que seré redirigido.<br>- Al confirmar, se abre una nueva pestaña con la URL externa de la convocatoria del gobierno. |
| **HU-10**| **Perfil Postulante** | Postulante | Como **postulante**, quiero **ver mi información de perfil** para **asegurarme de que mis datos estén actualizados**. | - Mi perfil muestra mi nombre, email y rol.<br>- Puedo ver mi foto de perfil (avatar). |
| **HU-11**| **Perfil Postulante** | Postulante | Como **postulante**, quiero **subir y gestionar mi CV** para **tenerlo listo para mis postulaciones**. | - Mi perfil tiene una sección para subir mi CV en formato PDF.<br>- El sistema guarda el archivo para futuras postulaciones. |
| **HU-12**| **Perfil Postulante** | Postulante | Como **postulante**, quiero **ver el estado de todas mis postulaciones** para **hacer seguimiento de mis procesos de selección**. | - Mi perfil muestra una lista de todas las ofertas a las que he postulado.<br>- Cada postulación tiene un tracker visual que indica el estado (Postulado, En Revisión, Entrevista, etc.).<br>- El sistema diferencia visualmente las postulaciones rechazadas. |
| **HU-13**| **Comunidad (Feed)** | Postulante | Como **postulante**, quiero **publicar mis experiencias en un feed comunitario** para **compartir información y conectar con otros usuarios**. | - Existe un formulario para escribir y enviar una publicación.<br>- Puedo añadir una calificación (de 1 a 5 estrellas).<br>- Si añado una calificación, mi publicación se vuelve anónima.<br>- Puedo mencionar empresas en mis publicaciones usando `@`. |
| **HU-14**| **Comunidad (Feed)** | Sistema | Como **sistema**, quiero **moderar el contenido del feed** para **prevenir la publicación de lenguaje inapropiado y mantener un ambiente seguro**. | - Antes de publicar un post, una IA analiza el contenido.<br>- Si el contenido es marcado como inseguro, la publicación se bloquea.<br>- El usuario recibe una notificación explicando por qué su post fue rechazado. |
| **HU-15**| **Gestión de Ofertas** | Empresa | Como **empresa**, quiero **ver un dashboard con métricas clave** para **entender el rendimiento de mis ofertas de un vistazo**. | - El dashboard muestra tarjetas con el número de ofertas activas, nuevos postulantes y entrevistas agendadas.<br>- Existe un acceso directo para crear una nueva oferta.<br>- Se muestra una lista con mis ofertas más recientes y sus estadísticas. |
| **HU-16**| **Gestión de Ofertas** | Empresa | Como **empresa**, quiero **crear una nueva oferta laboral** para **atraer talento a mi organización**. | - Tengo acceso a un formulario para crear ofertas (privadas por defecto).<br>- El formulario solicita título, modalidad, descripción, requisitos y condiciones.<br>- Al publicar, la oferta aparece en mi lista de ofertas y en la página pública. |
| **HU-17**| **Gestión de Ofertas** | Empresa | Como **empresa**, quiero **ver y gestionar mis ofertas publicadas** para **tener un control centralizado de mis vacantes**. | - Una tabla muestra todas mis ofertas con su estado, número de postulantes y acciones.<br>- Puedo buscar ofertas por puesto.<br>- Cada oferta tiene un menú de acciones (Ver postulantes, Editar, Eliminar). |
| **HU-18**| **Gestión de Ofertas** | Empresa | Como **empresa**, quiero **ver y gestionar a los postulantes de mis ofertas** para **evaluar a los candidatos**. | - Desde mi lista de ofertas, puedo acceder a la lista de postulantes de cada una.<br>- La lista muestra el nombre, avatar y fecha de postulación de cada candidato.<br>- Puedo hacer clic para ver el CV de un candidato en un visor de PDF integrado. |
| **HU-19**| **Gestión de Ofertas** | Empresa | Como **empresa**, quiero **editar y eliminar mis ofertas publicadas** para **mantener la información actualizada o cerrar procesos**. | - En mi lista de ofertas, cada una tiene opciones para "Editar" y "Eliminar".<br>- Editar me lleva a un formulario pre-rellenado con la información de la oferta.<br>- Eliminar abre un diálogo de confirmación para prevenir borrados accidentales. |
| **HU-20**| **Perfil Empresa** | Empresa | Como **empresa**, quiero **ver y editar el perfil de mi organización** para **presentar una imagen atractiva a los postulantes**. | - Puedo ver el nombre, email, descripción y sitio web de mi empresa.<br>- Puedo editar esta información a través de un modal.<br>- Puedo subir o cambiar el logo de mi empresa. |
| **HU-21**| **Misty X (Premium)** | Postulante | Como **postulante**, quiero **suscribirme a Misty X** para **acceder a asesorías de empleabilidad 1:1**. | - La página Premium muestra los beneficios de Misty X y un botón para suscribirse.<br>- Al hacer clic, soy dirigido a un formulario de pago simulado dentro de la app para una suscripción mensual.<br>- Al "pagar", recibo una notificación de éxito y acceso a las funcionalidades premium.<br>- Una vez suscrito, puedo agendar una asesoría viendo una lista de horarios disponibles.<br>- Al seleccionar un horario, puedo elegir entre los mentores disponibles para esa franja.<br>- Solo puedo tener una asesoría agendada a la vez. Si ya tengo una, la lista de horarios desaparece.<br>- Mi sección "Mis Asesorías" muestra el mentor, fecha y hora de la sesión.<br>- Hay un botón para unirse a la reunión (deshabilitado hasta que el asesor cargue el link).<br>- Hay un botón para cancelar la asesoría, que la elimina y vuelve a mostrar los horarios disponibles. |
