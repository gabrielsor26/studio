# **App Name**: Buscador Laboral

## Core Features:

- Role-Based Login: User authentication with role selection (User/Company).
- Profile Display: User profile display showing basic information (avatar, name, email).
- Main Navigation: Main navigation to switch between 'Opportunities', 'Feed', and 'My Profile' sections.
- Opportunity Listings: Job listings with key information (position, company, type, sector).
- Sector Filtering: Filter job listings by sector ('Public', 'Private', 'All').
- AI Content Moderation: AI moderation tool to detect and block inappropriate language before a post is published on the feed using Gemini.
- Keyword Search: Search jobs by position, company, or keyword.

## Style Guidelines:

- Primary color: Soft blue (#1E9DF1) for a professional and trustworthy feel, mirroring the provided theme.
- Background color: Light gray (#F7F8F8) for a clean, modern interface, maintaining consistency with the reference theme.
- Accent color: Complementary light blue (#E3ECF6) for interactive elements, drawn from the example theme for coherence.
- Font: 'Open Sans' (sans-serif) for clear and readable text, consistent with the example's design approach.
- Simple, outlined icons for intuitive navigation.
- Layout: Follow the general layout exemplified in the provided style to keep maximum visual consistancy.
- Subtle transitions for a smooth user experience.
